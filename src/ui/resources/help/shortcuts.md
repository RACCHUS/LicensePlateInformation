# Keyboard Shortcuts

Quick reference for all keyboard shortcuts in the application.

## Mode Switching

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+0` | Switch to All States mode |
| `Ctrl+Shift+1` | Switch to V3 mode |
| `Ctrl+Shift+2` | Switch to Express mode |
| `Ctrl+Shift+3` | Switch to I95 mode |
| `Ctrl+Shift+4` | Switch to OOSV3 mode |
| `Ctrl+Shift+5` | Switch to PlateType mode |

## Navigation

| Shortcut | Action |
|----------|--------|
| `Ctrl+F` | Focus search / Search all states |
| `Ctrl+G` | Jump to state dialog |
| `←` Left Arrow | Previous plate image |
| `→` Right Arrow | Next plate image |

## View Controls

| Shortcut | Action |
|----------|--------|
| `Ctrl+1` | Toggle state panel visibility |
| `Ctrl+2` | Toggle quick info bar |
| `Ctrl++` | Zoom in (images) |
| `Ctrl+-` | Zoom out (images) |
| `Ctrl+0` | Reset zoom |
| `F11` | Toggle full screen |

## Application

| Shortcut | Action |
|----------|--------|
| `F1` | Open help / User guide |
| `Ctrl+/` | Show this shortcuts reference |
| `Ctrl+,` | Open settings |
| `Ctrl+E` | Export state data |
| `F5` | Refresh database |
| `Alt+F4` | Exit application |

## Tips

- Most shortcuts work from anywhere in the application
- Mode switching is instant - no confirmation dialogs
- The search shortcut (`Ctrl+F`) focuses the search input and selects all text
